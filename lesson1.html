2. База знань: Породи деревини

Карпатська смерекаДеревина світлого кольору, з маленькими світлими сучками. Кращий варіант «ціна-якість» для вагонки, блок-хаусу, фальшбрусу, дошки підлоги.
Сосна звичайнаЖовтуватий відтінок із великими сучками, міцніша текстура. Відмінний матеріал для дахових конструкцій, дошки на підлогу.
ЛипаСвітлий колір, без сучків, добре переносить перепади температури, не виділяє смоли. Ідеально для бань та саун.
МодринаВисока стійкість до гниття, вологи, перепадів температур. Найкращий варіант для терас, садових доріжок, пірсів.

Переваги Смереки над Сосною

🌲 СМЕРЕКА

  • Світліша деревина, менше сучків.
  • Злегка жовтіє з часом, зберігає природність.
  • Більш однорідна структура.
  • Легше піддається обробці.
  • Менш схильна до деформації.

🌲 СОСНА

  • Жовтуватий відтінок, великі сучки.
  • З часом набуває насиченого жовтого кольору.
  • Більш виражена текстура.
  • Потребує ретельнішої обробки через смолу.
  • Більш схильна до деформацій.
Висновок: Для оздоблення стін та стель смерека є більш естетично привабливою та практичною.
